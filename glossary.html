A — E
ORGANIZATION
Alibaba(杭州)

Qwen シリーズの開発元。2026年4月20日に Qwen3.6-Max-Preview をリリース。下層オープン × 上層クローズドの二層戦略で、Red Hat / Android / Databricks の AI 版モデルを実践。

ORGANIZATION
Anthropic

Claude シリーズの開発元。2025年3月の OSTP 提言書で「late 2026 / early 2027 に "powerful AI" 登場」という公式予測を公開。AWS から最大 5GW の計算容量を確保。

HARDWARE
Ascend 910C / 950 / 960 / 970

Huawei の AI チップシリーズ。910C は 2026年 60万ユニット生産予定。960 で 2027年 Blackwell 追従、970 で 2028年 4 ZettaFLOPS FP4 を目標とするロードマップ。

HARDWARE
Atlas 950 SuperPoD / SuperCluster

Huawei の AI スーパーコンピュータ。SuperPoD は 8,192 チップで FP4 16 EFLOPS。SuperCluster は 64台束ねて 520,000+ チップ、524 EFLOPS FP8。xAI Colossus を処理ユニット数で 2.5倍上回る世界最大規模。

BENCHMARK
Arena(Chatbot Arena)

人間による投票でモデルを比較する評価基盤。2023年 GPT-4 は中国最上位に 300pt+ の差を付けていたが、2026年4月時点では 39pt 差(2.7%)まで縮まった。

HARDWARE
Blackwell

Nvidia の最新世代 GPU。7nm プロセス歩留まり 60-80%。Ascend 910C と比べダイ面積あたり性能で優位。

ALGORITHM
CANN / MindSpore

Huawei Ascend 上のソフトウェアスタック。CUDA + PyTorch との実運用上の成熟度差が、"Ascend時代" の成否を左右する。

ORGANIZATION
CAISI(Centre for AI Standards and Innovation)

米国の AI 標準化機関。中国モデル(DeepSeek 等)を「adversary AI」と位置付けたレポートを公開。

HARDWARE
Colossus(xAI)

xAI の大規模スパコン。Atlas 950 SuperCluster に処理ユニット数で 2.5倍追い抜かれる見込み。

MODEL
DeepSeek R1 / V4

DeepSeek の推論モデル / 次世代。V4 は Huawei Ascend 上で動作するよう最適化され、Nvidia / AMD への早期アクセスを拒否。1兆パラメータ MoE、1M トークンコンテキスト(Engram)が噂される。

CONCEPT
Embodied AI(具現化AI)

身体を持つAI。中国の産業ロボット設置台数は米国の9倍(29.5万 vs 3.4万)。将来的に3億台ヒューマノイド導入も議論される。AGIへの鍵である可能性も論じられる。

ALGORITHM
Engram(条件付きメモリ)

DeepSeek V4 の 1M トークンコンテキストを支えると噂される機構。リポジトリ全体解析、エージェント長期記憶などの新用途を開く。

F — K
ORGANIZATION
Figure AI / Apptronik / UBTech

ヒューマノイドロボット企業。Figure は家庭展開を 2025-26 のパイロット範囲で。UBTech は中国自動車工場で最初のヒューマノイド・チーム展開済み。

MODEL
GPT-5.4 / Claude Opus 4.6

2026年4月時点の米国フロンティアモデル。SWE-Bench Pro / HLE で K2.6 と並び、一部の欄で追い抜かれた。

BENCHMARK
HLE(Humanity's Last Exam)

人類最後の試験。最高難度の評価基盤のひとつ。K2.6 がここでも首位欄に並んだことで「追い抜く」という動詞が初めて書き換えられた。

ORGANIZATION
Huawei

Ascend チップ / Atlas スパコンの開発元。2026年に Ascend ライン合計 1.6M ダイの供給を計画。Alibaba、Tencent、DeepSeek が主要顧客。

ALGORITHM
Kimi Delta Attention (KDA) / Kimi Linear

Moonshot の線形系アテンション設計。Transformer の二次計算量を線形に近づけ、長文脈 2.9倍・デコーディング 6倍高速化の可能性。

MODEL
Kimi K2.5 / K2.6 / K3

Moonshot のフラッグシップ。K2.5→K2.6 は 2ヶ月。K2.6 は 12時間自律実行 / 300並列 / 4,000+ ツール呼び出し。K3 は 3-4兆パラメータ規模が内部ロードマップに。

L — P
MODEL
LLaMA

Meta のオープンウェイトモデル。「フロンティアに肉薄」したが、特定ワークロードで「首位を獲った」ことはなかった。K2.6 がこの境界を初めて越えた可能性。

ALGORITHM
MLA(Multi-head Latent Attention)

DeepSeek の KV キャッシュ圧縮。長文脈処理のメモリ負荷を激減させる。MoE との組み合わせで効率を極限化。

BENCHMARK
METR

AI の時間的タスク実行能力を評価する研究所。外挿により「50%信頼性での16時間タスク実行」を 2027 末の現実的到達点と見積もる。

ORGANIZATION
Moonshot(北京)

Kimi シリーズの開発元。2026年4月20日、K2.6 をオープンウェイトでリリース。長時間エージェント実行の領域で、オープン×フロンティアを初めて成立させた可能性。

ALGORITHM
Muon Optimizer

Kimi K2 が採用した新しい最適化アルゴリズム。従来より効率的に大規模モデルを訓練できる。

MODEL
Muse Spark

Artificial Analysis ベンチで 1位。誰も注目していなかったラボが突然トップに立った例。「突然出てくる最強モデル」現象の象徴。

HARDWARE
Nvidia H100 / H800

米国フロンティアラボの主力 GPU。H800 は中国輸出向け。DeepSeek は R1 で H800 を使っていたが、V4 で意図的に Ascend へ移行。

HARDWARE
PJM

米国東海岸の送電網運営組織。容量市場約定価格が 2024-25 の $28.92/MW から 2026-27 に $329.17/MW へ 10倍以上に高騰。電力ボトルネックの象徴。

CONCEPT
Powerful AI

Anthropic Dario Amodei が "Machines of Loving Grace" で描くビジョン。ノーベル賞受賞者クラスの知性を多数専門領域で発揮する水準。「データセンターの中に天才の国」。

Q — U
MODEL
Qwen3.6-Max-Preview / 35B-A3B

Alibaba の 2026年4月20日リリース。Max はクローズド、4日前の 35B-A3B はオープン。二層戦略の象徴。Qwen オープン利用は 1年で 1.2% → 30% へ。

ORGANIZATION
SMIC

中国の大手半導体ファウンドリ。7nm プロセス歩留まりは 5-20%。EUV 露光機にアクセスできないことが、Huawei Ascend の物理的な天井。

BENCHMARK
SWE-Bench Pro

ソフトウェアエンジニアリング能力を評価する高難度ベンチ。K2.6 が GPT-5.4 / Claude Opus 4.6 と横並びになった主要指標のひとつ。

ORGANIZATION
Stanford AI Index 2026

スタンフォードが毎年発行するAI統計。2024年の米国民間AI投資 $2,859B、中国 $124B 等の数字ソース。性能差 2.7%、人材流入 −89% もここに。

ORGANIZATION
Stargate

米国主導の巨大データセンター投資プロジェクト。中東展開も進む。Global South 争奪戦の米国側の切り札。

ORGANIZATION
天河 / 神威

2010年代に中国が世界最速の座を争った HPC シリーズ。Atlas 950 SuperCluster は、その「物量で HPC を獲る」戦略の AI 時代への延長線。

あ — わ
CONCEPT
AIマネージャー

部下10人の課長から、エージェント1,000体を監督するオペレーターへの転換。持久戦AIが実運用化する 2026-27 年に出現すると予想される役割。

CONCEPT
二層戦略

Alibaba が Qwen で実行中。「下層はオープンで普及、最上層はクローズドで収益化」。Red Hat / Android / Databricks の AI 版。

CONCEPT
持久力パラダイム

1回の応答の質ではなく、一晩中壊れずに働き続けられるかを問う評価軸。IQ 競争からアスリート競争への移行。K2.6 が提示した「新しい軸」。

CONCEPT
地図の「種類」が変わった

2024年までの「単一ランキング」が崩れ、用途別に複数の地図が並立する状態へ。長時間エージェント、フロントエンド生成、数学、視覚、多言語、安全性 — それぞれで首位が違う。
